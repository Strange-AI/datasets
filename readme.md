# Datasets

人工智能智能时代，依旧无法离开数据。奇异AI的工作者们收集了大量的数据来源，甚至在官网开辟了一块领域让大家方便的快速进入各大数据集入口。我们从以下领域出发收集来包含18大领域，近300种各类数据集。但其中有许多还无法完全包含进来，如果你有好的数据集来源，再可用的情况下可以给我们send PR。


**Update**: 最新跟进的大规模面向深度学习数据集分类收集
